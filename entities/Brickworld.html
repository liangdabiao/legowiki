Brickworld（砖块世界）

美国最大的乐高粉丝展会之一，每年在芝加哥等地举办，汇聚全球 AFOL（成年乐高粉丝）展示 MOC 作品、参与搭建挑战和社区交流活动。

展会历史

Brickworld 始创于芝加哥，经过多年发展已成为美国规模最大、最具影响力的乐高粉丝聚会之一。展会通常持续多天，其中前半段为参展者搭建和准备期（“setup days”），后半段向公众开放。2020年至2021年间，受新冠疫情影响，展会被迫中断，期间曾举办线上虚拟版本 Brickworld Virtual。2022年6月，Brickworld 在中断三年后成功回归线下，标志着乐高社区线下活动的重要复苏。

展会活动内容

Brickworld 芝加哥展会涵盖以下核心活动：

  1. MOC 作品展示：参展者搭建的原创乐高模型展示，涵盖城市景观、科幻场景、大型雕塑等多种主题
  2. 搭建挑战赛：仿照乐高大师（LEGO Masters）节目形式的限时搭建比赛
  3. 专题讨论会：由知名搭建者和乐高大师参赛者主持的分享和交流活动
  4. 公众见面会：与乐高大师参赛者和社区知名创作者的互动环节
  5. 现场直播：YouTuber 在展台前进行实时直播，实现全球同步曝光
  6. 周边商品：展会限定 T 恤、抽绳背包等纪念品销售

展会特色

  • 多种展示台高度：2022年起采用三种不同高度的展示台，为参展者提供更多分层陈列空间，同时改善观众的观赏体验
  • 首次参加者比例高：2022年展会中492名为首次参与者，约占注册总数近45%，反映了展会的持续吸引力
  • 多渠道宣传推广：通过电台、电视（国家地理、ESPN）、社交媒体（YouTube、HULU）等多平台投放广告
  • 安全措施：提供大量免洗洗手液等卫生防护物资

相关人物

  • Mark Larson：展会负责人，被称为”Brick-xecutive”，同时是活跃的乐高搭建者，曾参与 Eurobricks 合作项目，包括人仔比例动画作品和姜饼屋村庄，其代表个人作品为 Fabuland 飞艇
  • Maria 和 Philip Straatsma：BrickNerd 校友，乐高大师参赛者，在展会上活跃参与各类活动

Brickworld 在北美乐高展会生态中的地位

Brickworld 与 BrickFair、BrickCon、BrickFest 并列为北美四大乐高粉丝展会。与乐高集团官方举办的 LEGO Fan Media Days 或 Inside Tour 不同，Brickworld 完全由粉丝社区自发组织运营，是典型的”社区驱动型”（community-driven）活动。这种模式赋予了展会独特的草根气质和高度自由度——参展者不受商业品牌限制，可以展示任何题材和风格的 MOC 作品。Brickworld 芝加哥主会场所在的文艺复兴酒店及会议中心拥有约10万平方英尺的展厅空间，这一规模使其在北美同类活动中名列前茅。展会采用”搭建日 + 公众日”的双阶段模式，搭建日仅对已注册的参展者开放，便于进行展品布置和技术交流；公众日则向普通观众售票开放，承担了乐高文化推广和社区招新的重要功能。

大型乐高展会的物流与技术挑战

举办一场千人级别的乐高展会涉及复杂的后勤保障工作。首先，大型 MOC 作品的运输是核心难题——许多展品包含数千甚至数万零件，重量可达数十公斤，需要参展者自行打包、运输并在现场重新组装。展台的标准化和稳定性至关重要，Brickworld 2022年引入的三种高度展示台设计正是对这一需求的回应：较低的展示台适合大型景观类作品的全景展示，中等高度适合标准 MOC 的独立陈列，较高的展示台则有利于观众近距离观赏精细细节。其次，展会的电力供应和网络基础设施也需精心规划，尤其是当 YouTuber 进行现场直播时，稳定的网络连接成为刚性需求。展品安全同样是不可忽视的环节，人群密集的公众日期间需要防止展品被触碰或损坏，许多参展者会使用透明亚克力罩或设置隔离带进行保护。

疫情后乐高展会复苏与社区转型的深远影响

2020年至2022年间的新冠疫情对全球乐高展会生态造成了深远冲击，几乎所有大型线下展会被迫中断或转为线上。Brickworld Virtual 是这一时期的典型产物，尽管它维持了社区的连续性，但无法替代线下展会的面对面交流体验和展品实物观感。Brickworld 芝加哥2022年的回归具有标志性意义：超过1,100名注册者中有近45%为首次参加者，说明疫情不仅没有削弱社区活力，反而因积压的参与需求催生了新一轮增长。展会引入 YouTuber 现场直播等数字化转型举措，反映出后疫情时代乐高社区”线上+线下”融合发展的新趋势。这一趋势对全球 AFOL 社区具有借鉴意义——线下展会依然不可替代，但数字化工具可以显著扩展其影响范围，让无法亲临现场的全球粉丝也能实时参与。
