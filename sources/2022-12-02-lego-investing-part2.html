乐高投资指南（第二部分）—— 深度分析与策略

一句话总结：深入分析乐高作为投资品的表现数据、市场趋势和专业投资策略。

完整内容

BrickNerd很高兴呈现这篇关于乐高投资短篇系列的第二篇文章，我们将分析长期持有乐高套装是否是获利的可靠策略。

Safe build by Tom Gion and boxes by RAWF8, VIA ENVATO ELEMENTS

此前在BrickNerd…

在本投资系列的第一部分中，我通过分析2005年至2021年间退市的所有500块零件以上套装的价格涨幅，回顾了乐高投资的回报。826套样本的年投资回报率达到了健康的11.7%，与Victoria Dobrynskaya在一份学术研究中发布的11%回报率一致。

Image by rthanuthattaphong, via Envato Elements

然而，尽管回报率令人印象深刻，但样本只计入了已售出的套装，并没有关注那些仍挂牌待售的套装。当分析当前在BrickLink上挂牌出售的商品时，发现了一些值得关注的问题。在第二部分中，我们将揭示这些问题，回答这个问题：“货架上还剩下什么？“

4. 库存周转率

不仅仅是利润

“投资回报率”（ROI）和”拆卖价值”（POV）分别是乐高投资者和BrickLink卖家使用的盈利能力指标。但这两种方法都没有考虑实现利润所需的时间长度。

为此，我们需要使用库存周转率（有时称为存货周转率）。它是衡量库存效率的指标，听起来很枯燥，但对企业（当然还有乐高卖家）来说非常有用。它显示一个组织的库存放在仓库或货架上直到售出需要多长时间。计算方法简单，可以使用公开信息进行计算。

企业的库存周转率

从乐高集团2021年年度报告中的数据，我们可以估算出乐高集团的库存周转率为76天，或更宽泛地说，2个半月。

平均而言，从原材料运抵乐高工厂到你我在商店购买成品（或运送到亚马逊等其他乐高零售商的仓库），需要2个半月的时间。

Image by krisprahl, via Envato Elements

英国大型超市Tesco的库存周转率低得多，只有15天，这是可以预期的，因为他们只是买卖成品，不像乐高集团需要自己制造。威士忌酒厂则处于另一端，需要以年来衡量库存周转率。一瓶12年的艾莱岛单一麦芽威士忌在桶中陈酿12年，然后才卖给口渴的顾客（我就是其中之一）。

准时制生产

企业一直在努力减少持有的库存量，这可以追溯到古老的会计格言：“周转是虚荣，利润是理智，现金才是王者！“如果销售和利润保持不变，但公司可以通过提高效率将库存减少1亿英镑，那这1亿英镑就直接转化为现金。他们还将受益于更少的仓库空间和相关的成本节约。这些效率创造了全球”准时制”体系，并体现为消费品价格的降低。

Image by tampatra, via Envato Elements

但这与乐高投资有什么关系？很明显，无需任何分析就能看出，乐高投资者更像是威士忌酒厂，而不是零售商或制造商。库存存放数月，更可能是数年之后才能售出并实现利润。起初这可能并不重要，但当你考虑到存储空间、通货膨胀和长期占用资金等因素时，就必须仔细考虑这个问题。

5. BrickLink的供需关系

BrickLink上的库存周转率

BrickLink是少数几个你可以计算平台上每种商品库存周转率的地方：每块砖、板、砖、人仔、 jumper、得宝砖、科技销、贴纸和套装。例如，10264 街角修车厂在2021年退市，目前在BrickLink上售价为184英镑，比原零售价160英镑高出15%。

在6个月期间售出了173套10264 街角修车厂，换算下来年销售量为每年346套（需求）。目前BrickLink上有688套待售（供应），这意味着需要2年时间才能售出所有列出的套装。

这是非常有用的信息。如果你今天在BrickLink以平均价格列出街角修车厂，平均可以预期要等待2年才能卖出。你可能运气好下个月就卖了，但也可能不走运，需要4或5年才能卖出。

当我们用库存周转率比较不同的在售产品时，它变得更加强大。

上表显示了四种不同颜色2x4基础砖的价格。橄榄绿砖的销售时间是浅灰砖的六倍多，但目前价格是浅灰砖的两倍。这是怎么回事？

在有效市场中，我们预计橄榄绿砖的价格会快速下跌，因为这个价格的需求疲弱。但鉴于BrickLink定价机制，价格下跌可能需要一段时间。使用历史数据定价会创造一个自我实现的循环，减缓价格反映自然供需平衡的过程。

羊群定价

关于羊群定价这个话题，还可以写一整篇文章，但就目前而言，我们只想说BrickLink价格存在一些滞后效应，而库存周转率可以揭示这些低效率。

一位勇敢的BrickLink店主可能会决定根据库存周转率（我们的供需代理）完全重新定价。但这样做，他们可能会面临这样的风险：顾客只买降价的零件，忽略涨价的零件，选择从其他商店购买那些涨价零件。

从本文所做的工作，我在非常高的层面上得出结论：

  • 需要3年以上才能卖出的零件或套装可能定价过高。
  • 需要1到3年才能卖出的零件或套装定价可能恰到好处。
  • 不到1年就能卖出的零件和套装”卖得飞快”，可能定价过低。

这不是全面分析，但对我们来说是一个有用的晴雨表。

6. 冷藏库存

30年前的今天

第一部分的样本确定了2021年退市的96套套装，平均投资回报率为+9.5%。然而，其中三分之二的套装目前正在亏损出售。此外，43%的套装库存周转率超过5年，27%的套装周转率超过10年！

一个极端的例子是75254 AT-ST 突袭者，目前在BrickLink上每年销售106套（需求），但有3,217套可供出售（供应）。需要超过30年才能售完这些库存，最后一套将在2050年代初售出。这绝对是一个纪元！（这可能是因为由于一位女演员离开制作，这套中有独家星球大战人偶，但似乎很多人都有把这套放在货架上收藏的想法。）

image via brickset

往回走30年，在1990年代初，6086 黑骑士城堡就在你当地商店的货架上，旁边是Nirvana、Primal Scream、REM、Massive Attack，当然最重要的是My Bloody Valentine的黑胶唱片和磁带。30年真的很长。

75254 AT-ST 突袭者并非孤例。以下是2021年退市类别中销售最慢的20套套装：

目前BrickLink上有15,612件这些慢销套装可供购买，但年需求仅为678件，这意味着按当前速度需要23年才能售完所有这些商品。

简单出路？

纵观所有2021年套装，我们可以更全面地看到投资回报率与库存周转率之间的关系。这种关系并不完美，下图中没有一条从左上角到右下角清晰的直线。

但我们可以说，如果一套套装目前盈利出售，它也卖得快——持有石灰绿点组套装的投资者处于有利位置。他们唯一的决定是现在兑现还是持有，期待进一步上涨。

其次，可以说，如果一套套装销售缓慢，它几乎肯定在亏损出售（除了21043 旧金山等少数例外）。持有珊瑚红点组套装的投资者将面临一些巨大挑战，因为供应远大于需求。

也许拆分这些套装以收回最大金额是最好的方法，而不是让它们占据货架空间，但这也有其自身问题，特别是如果投资者还没有开设BrickLink零件店的话。

亮橙点组是一个不太符合模式的模糊群体。平均库存周转率为2.3年，更接近盈利组（1.5年），但其-11%的投资回报率更接近慢销组（-22%）。鉴于这里供需平衡更好，最好的策略可能是持有，期待未来价格上涨。

按退市年份划分的库存周转率

2021年退市套装的平均库存周转率为3.30年，仍然是2007年至2019年退市套装平均1.64年的两倍。2020年退市套装表现稍好，为2.40年，但仍远高于历史平均水平。

对这些数据的一种解释是，近年来乐高投资变得更加普遍，导致供应增加，库存周转时间延长。

但这并不简单，这可能是套装退市后立即出现的正常模式。总的来说，我对这些增长的数据感到担忧，并将密切关注即将在2022年退市的套装的表现。

7. 未来回报

未来学家走进酒吧

预测未来是一件棘手的事情。我做出过斩钉截铁、狂妄自大的预测，但结果很快就让我看起来像个大奖章萝卜，这种情况发生次数不说零也不多了。但这并不意味着尝试不好玩，如果预测置于正确的背景下，并且了解支持性分析是如何进行的，它们可以非常有用和信息丰富。

Image by andriymedvediuk, via Envato Elements

为了更好地量化上面强调的库存周转问题，并帮助我们了解乐高投资未来可能的表现，我们需要为未售出套装创建一个比较投资回报率。我们已经看到，我们样本中已售出套装的回报为11.7%，我们称之为”历史投资回报率”。

我们新的比较指标**“未来投资回报率”**的计算方法相同，只是不是按已售出套装加权，而是按BrickLink上可供出售的套装加权。它有效地显示，如果BrickLink上列出的所有商品今天都以过去6个月的平均售价出售，投资回报率会是多少。

总的来说，图表支持库存周转部分强调的问题。如果今天列出的所有套装都按当前价格出售，它们的投资回报率将为0%，盈亏平衡。显然，这与历史销售带来的11.7%回报率有很大不同。

对于2007年至2016年退市的套装，历史投资回报率和未来投资回报率之间的差异微乎其微。然而，从2017年开始开始下降，到2021年累积达到：历史投资回报率+9%降至未来投资回报率-10%。整体未来投资回报率为0%受到2021年数据的严重影响，该年份占权重的近一半（47%）。我不能肯定地说乐高投资正进入低迷期，但这些结果不能被解读为积极的。

是的，未来大多数套装的价格可能会上涨，但我们必须记住，投资回报率是年化数据。一套投资回报率为10%的套装，在接下来的12个月里需要再上涨10%才能”保持原位”，维持10%的投资回报率。

Image by leungchopan, via Envato Elements

再次说明，我不想做出任何绝对的结论，只是说这些结果结合库存周转率数据，至少是一个警告信号，表明我们看到乐高投资回报正在下滑。

如果真是这样，我们可以推测是什么潜在原因推动了下滑。从供应方面看，可能越来越多人投资套装，以便将来出售。库存周转率数据在一定程度上支持了这一点。但从需求方面来看，乐高集团生产的套装数量和质量的增加，可能比过去占据了我们”乐高支出”中更高的比例。（感谢我们的主编Dave在我们计划这篇文章时提出了这个假设。）

我认为，要确定无疑，唯一的方法是在12年后重复这个练习，看看2017年至2021年的未来投资回报率数据是否有向上移动，以及2022年套装在退市后第一年的表现如何。

8. 结论

六个要点

我非常感谢有机会撰写这篇文章。这是一个迷人的话题，包含了大量有趣的数据，我很乐意谈论提出的任何观点，并接受任何建设性的批评。

但就目前而言，我想说这是乐高投资的六个关键要点：

  • 在我们对2005年至2021年间退市的826套（超过500块零件）套装样本中，乐高投资的回报率每年达到非常健康的11.7%，并且这些回报在各年份套装中保持一致。
  • 然而，二级市场的规模极小，使得与股票、债券和其他投资的比较没有意义，特别是考虑到乐高投资需要占用物理空间和时间。
  • 库存周转率是BrickLink上一个有用的指标——对套装和零件卖家都适用。它也可以作为供需代理，揭示平台上定价的低效率。
  • 2021年退市套装的整体投资回报率为9.5%，但其中三分之二的套装目前由于供应过剩而亏损出售。表现最差的套装库存周转率超过20年。
  • BrickLink上当前在售套装的投资回报率为0%，盈亏平衡（假设所有列出套装今天都以过去6个月的平均价格出售）。
  • 最后，“历史投资回报率”11.7%与”未来投资回报率”0%之间的差距在过去五年中不断扩大，表明乐高投资可能正进入低迷期。12个月后重新分析将有助于我们得出更稳定的结论。

最后想法

我强烈推荐Jerry Z. Muller的一本书，书名是**《指标的暴政》**。鉴于本文的数字内容，这听起来可能有悖直觉，但他的话表达了我一段时间以来在使用数据和关键绩效指标（KPI）方面的感受。

Image by FoToArtist_1, via Envato Elements

第一点是，指标只应该作为在对情况进行全面评估或支持决策过程中的一项证据。这里我们的两个投资回报率指标，11.7%（历史）和0%（未来）就是如此。这两个指标都不一定是”正确答案”，即使在高位层面它们都有用，但它们只是平均值，不是保证。如果你是乐高投资者，会有许多对你而言独特的因素影响你的决策，最终影响你的成功。

第二点是，如果你把所有时间都花在关注你能测量的东西上，你会忽略哪些你无法测量的重要事物？这回到了乐高投资与爱好其他方面并行的个人享受。我进行了分析，像写严肃市场评论一样写了这篇文章。尽管很有趣，但我怀疑对大多数人来说，乐高投资真的不是那么回事。

Image by Christopher Clarke

你无法衡量买两套套装，一套拼搭另一套收藏，然后看到它在二级市场飙升，带来不错的利润再去买更多乐高的兴奋。你无法衡量有多少成年人以”我以后可以卖掉它”为由给自己买套装。你无法衡量那些主要表达对爱好热爱就是收藏完美密封盒的乐高粉丝的快乐。

在这一点上，最后我只想说…享受你所做的，祝你投资愉快！

研究方法

对于那些对本系列文章中使用的方法感到好奇的人，以下是得出上述结论所考虑的因素和计算。

  • “基本回报”是退役套装在BrickLink上2022年9月前6个月的平均售价与Brickset上列出的退役乐高建议零售价（RRP）之间的差额。
  • 对每套套装的建议零售价应用20%的调整，以反映乐高投资者通过折扣购买的可能性。对平均售价应用15%的降低，以反映平台费用和其他杂项销售成本。
  • 不对投入的时间或存储空间进行调整。
  • 对每套套装，调整后的基本回报然后年化以显示每年产生的回报。这允许不同年份退市的套装直接相互比较。
  • 然后，根据每套套装的BrickLink 6个月总销售额（英镑）对年化投资回报率进行加权，以显示按退市年份分组的平均投资回报率。然后基于相同的加权方法进一步聚合，得到本文引用的11.7%投资回报率。这有时也被称为”历史投资回报率”。
  • “未来投资回报率”的计算方法与历史投资回报率相同，不同之处在于权重基于2022年9月BrickLink上列出的商品数量，而不是过去6个月的销售量。
