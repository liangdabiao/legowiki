创造角色：Baron Julius von Brunk专访

BrickNerd专访多媒体乐高艺术家Baron Julius von Brunk，深入了解他从Nintendo变形金刚到巨型乐高头像的创意世界、定格动画制作和电子装置创作。

概述

本文最初发表于BrickNerd（2021年6月16日），是对多媒体乐高艺术家Baron Julius von Brunk的专访。Julius是一位居住在纽约市的平面设计师，以其标志性的Nintendo主题变形金刚、巨型乐高头像雕塑、定格动画以及各种电子装置乐高创作而闻名。采访涵盖了他的创作历程、艺术风格演变、标志性作品背后的故事以及未来计划。

完整内容

新鲜而难忘的情感——这是我们在任何艺术博物馆欣赏展品时获得的奖赏。在乐高画廊中，同样的”WOW”效果是通过明亮且发人深省的积木创作实现的，让我们感到惊讶并完全专注于搭建者不寻常的内心世界。

今天，这些强烈的印象几乎是有保证的！女士们先生们，请认识Baron Julius von Brunk，他是我们今天BrickNerd社区的特色嘉宾，一位多媒体乐高艺术家。准备好踏上一段令人兴奋的旅程，进入他丰富多彩的想象力，了解Baron von Brunk如何将他的积木王国变成一个真正的童话。

Ann： 嗨Julius！很高兴能采访到如此充满热情和才华的乐高搭建者！在引导我们穿越你壮观的乐高宇宙之前，能先介绍一下你自己吗？

Julius： 我是一名30多岁的平面设计师，在金融行业工作，目前住在纽约市。我在宾夕法尼亚州兰开斯特出生长大，从小收集变形金刚、星球大战，当然还有乐高积木等玩具。我从很小的时候就开始用乐高积木搭建。大约在2000年代初社交媒体开始时，我开始拍摄我的作品。现在我的摄影技术提高了，近年来我开始制作乐高定格动画。我计划最终发布一部完整长度的砖块动画视频。

Ann： Baron von Brunk和乐高是如何相遇的？乐高是如何变成你最爱的艺术媒介的？

Julius： 当我大约三岁的时候，在1980年代，我的父母会给我买各种各样的乐高套装——特别是太空和城堡主题。在90年代成长过程中，我会花大量时间做各种各样的原创作品，包括受视频游戏启发的模型——但当时受限于可用的颜色和零件种类。然后在2000年代，当乐高集团在其目录中添加了大量新颜色和零件时，这极大地帮助了我制作更精细和可活动的搭建。

后来，当我在2010年代搬到纽约追求数字艺术职业时，我的摄影技术大大提高，我将乐高搭建和摄影融入了我的艺术作品中。我从2014年开始使用单反相机，目前拥有两台尼康相机型号来进行所有拍摄。

Ann： 你能描述一下你个人独特的乐高创意艺术风格吗？你是如何形成和发展它的？

Julius： 关于我用乐高的独特艺术风格，最初我通过制作任天堂和世嘉相关的搭建为自己开创了一个细分领域，那还是在乐高推出视频游戏角色授权产品很久之前。值得注意的是在2011年底/2012年初，我创作了一些标志性的任天堂乐高作品，比如我的巨型超级马里奥3飞空艇和Game Boy变形金刚：这两个项目都在2012年5月的任天堂力量杂志中刊登。

从那里我继续走着类似的游戏和流行文化搭建之路，但当我开始用单反拍照时，开始强调艺术摄影。有一段时间，我做了一些极简主义摄影，在户外拍摄人仔照片，这在以前开车的时候经常做。这些照片经常在2015-2016年间发布在我的Instagram上。

随着时间的推移，我开始尝试乐高定格动画，大约从2017年左右开始。我的砖块动画的独特之处在于它们经常使用精细的场景和可活动的木偶，而不是简单的人仔动画。

Ann： 我可以看出Baron von Brunk喜欢扩展想象力的边界，拍摄令人惊叹的童话故事。那么你最喜欢的积木故事是什么？

Julius： 我建造大型NES控制器的故事是一个有趣且持续的故事。简单来说，我最初受”里普利信不信由你”时代广场博物馆的委托，为一次宣传活动建造一个巨大的蝙蝠侠蝙蝠车乐高雕塑，但遗憾的是项目被取消了，我被允许保留项目的所有零件。

我没有让它们浪费，而是决定在2012年剩余时间里用它们为自己建造一个5英尺（1.5米）大的功能性经典任天堂控制器雕塑。当我在2013年1月1日发布这个项目和它的视频时，作品走红了，我很快被美国任天堂联系，他们委托我为新Wii U游戏发布在洛克菲勒中心旗舰店搭建一个店面展示。我认为这是我最大的成就之一。

2015年我重新改装了控制器的部分，使其可以运行电脑游戏而不仅是经典NES游戏，然后在一次视频游戏展会上展示它让人们可以试玩。不幸的是，由于体积庞大和脆弱，运输极其困难，在往返展会的途中严重受损。随后我在2016年完全拆除了整个作品并从头重建，设计成可拆卸为多个单元以便更容易运输。目前的NES控制器（Mk.III版本）现在可以便携运输，有潜力在公共活动中相对容易地展示。

Ann： 你的媒体项目中有很多古怪而令人难忘的角色。你乐高作品中你最喜欢的角色是谁？

Julius： 绝对是我的爱丽丝人仔，它被用在我的乐高《爱丽丝梦游仙境》视频动画中，遗憾的是这个项目在过去几年一直停滞在制作中。我用爱丽丝角色制作了很多动画素材，但还没有发布最终项目。当最终发布时，我希望让爱丽丝角色成为Baron von Brunk创意宇宙中的主要标志！

在2016-2017年我做了全乐高《爱丽丝梦游仙境》砖块动画的预制作工作，但基本上贪多嚼不烂，低估了这个项目的难度。

我花了几个月时间制作一些动画并逐步取得进展，但由于当时生活中的主要障碍——比如工作或住房情况——无法集中精力完成它。正是因为这部电影的纯粹难度，我将其搁置，转而做更小的动画来建立我作为动画师的品牌。

Ann： 你著名的乐高头像真的强化了”你就是你所搭建的！“这个理念。这个项目绝对是你展示的面孔，也是一个很好的自我表达工具！巨型积木头像的想法是怎么来的？

Julius： 谢谢！长话短说，在2015年底我计划发布一本包含我最好乐高作品摄影的书，我最初建造头像雕塑作为封面的中心装饰。事实上，我最初的计划是让书的封面是一张由一队宇航员人仔组装头像雕塑的照片。

然后在2016年春天我逐渐开始制作头像，并出乎意料地在短时间内完成了。发布之后，我略微改进了它，使眼睛在按下头像后面的按钮时通过小LED灯闪烁。虽然我从未发布这本书，但我确实计划在不久的将来发布它，将头像雕塑纳入封面艺术中。

Ann： 说到搭建，你在建造任天堂变形金刚方面有着令人难以置信的经验值得一提！

Julius： 变形金刚是我从小最喜欢的卡通片和玩具系列之一，所以我在制作自己的定制变形玩具的功能性和变形周期方面确实有大量知识。我的很多乐高变形金刚受到特定变形金刚玩具型号的启发，特别是震荡波和声波——所以我经常使用它们的基本设计和功能作为我模型的基础。

我的第一个任天堂变形金刚是我2011年底的橙色光枪”Plasmashock”，从那里起我只会不断提高我的技能。到目前为止我最喜欢的模型是我的任天堂64”Ultra Hexacon”。在写这篇文章的时候，我计划在接下来的几个月内发布一个变形超级任天堂。

我是去年开始做的，完成了两个控制器和游戏卡带，但现在我从我的旧存储单元取回了经典视频游戏收藏，我打算用我真正的超级任天堂作为参考模型来制作我的SNES变形金刚，使其尽可能准确。

Ann： 换个话题，我很想了解你在YouTube频道上分享的定格动画制作过程。例如，你关于Shrek吃鸡块的故事非常有趣！

Julius： Shrek鸡块视频绝对是一个奇怪的故事，也是我最喜欢的动画之一，因为其纯粹的荒谬性！在2019年，我想发布几个小型、简单的动画来补充我的YouTube频道，所以在一次坐火车回家的路上，我在思考一些快速视频的一般想法。我的思路不知怎么让我产生了用乐高做Shrek吃鸡块的想法，从那里我继续写下想法并为视频做准备。

动画花了大约一个月的时间来做，因为让Shrek的嘴巴动是一个非常繁琐的过程；每次我张开他的嘴时，我必须在他嘴巴两侧插入酸绿色乐高零件来使他的头部保持在原位。我的其他乐高木偶动画更加流畅，因为头部更简单更轻。我还花了相当多的时间试图制作结尾的音乐，因为我不知道如何演奏任何乐器，所以只能用合成器数字合成音乐曲目。

为了避免浪费食物，Shrek每咬一口我都重复使用同一块鸡块，然后换成第二块切成碎块的鸡块。我最终吃掉了用于特写镜头的剩余鸡块。

在制作过程中，我还为一个我想做的潜在续集写了想法，但遗憾的是由于后勤问题没能完成。我的续集计划叫”Shrek是机器人”，Shrek木偶会有一个他是机器人的顿悟，然后撕开自己的内心发现机械零件和电线。我脑子里有一个大致的粗略想法，但没能完全将其实现。

Ann： 说到电线，什么启发了你的电子创作？设计、点亮并为你的想法注入生命需要多长时间？

Julius： 交通灯的想法是我在坐火车下班回家的路上想到的，当时我盯着显示站名的滚动LED标牌。我产生了一个使用Technic砖块和透明点的点阵图案制作发光乐高创作的想法。我在这个项目上花了相当多的时间，因为当时我对电子学知之甚少，必须一步一步地弄清楚一切。事实上，我那时还无法编程微芯片或使用Arduino，所以我通过稍微修改一个从爱好网站购买的模型火车控制器来创建主电路。

马赛克超级马里奥灯是大约同一时间的另一个类似项目，采用了相同的多色点阵图案的基本原理。这些灯稍微简单一些，使用中央的基本灯泡照明而不是程序化电路板。我最终在建造几年后的一次视频游戏展会上卖掉了大部分这些立方体灯。

至于迷你超级马里奥精灵，我是后来在2014年创建的，当时我的电子技术强得多，因此我整合了一个我创建的小微芯片电路来激活灯光和声音。在制作精灵之前我用这个基本概念做了实验，一旦我有了成功的源代码和原理图，我就继续建造了几个这样的小精灵在视频游戏展会上出售。

Ann： 你感觉在社交媒体和现实世界中分享你的作品如何？你是乐高展会和展览的活跃参与者吗？

Julius： 我通常在现实生活中得到很好的反馈。我在一个需要广泛背景调查的行业工作，所以通常我的雇主都熟悉我的网络存在。经常发生我的同事和主管告诉我说他们去了我的YouTube频道并喜欢我的视频！

我有些内向，所以不经常公开展示我的作品——但过去确实做过几次。值得注意的是在2013年春天，我在曼哈顿任天堂世界商店的玻璃展示柜中展示了我很多任天堂-乐高作品，为Wii U的乐高城市：卧底风云发布派对。

几年后，我在新泽西的一次视频游戏展会上展示了我的各种作品，令人惊讶地卖出了相当多的任天堂作品。更不用说我还让我的巨型NES控制器在一个展台上供人们试玩，因为我把它连接到了一台运行俄罗斯方块的电脑上。

Ann： 你在乐高搭建和艺术领域有很多成就。作为一位被认可的艺术家的你最自豪的是什么？你有未来的计划或梦想吗？

Julius： 坦白说，我最大的成就是在2013年在任天堂世界商店展示我的作品。如前所述，这是因为美国任天堂联系我关于我的巨型NES控制器。这次活动特别的地方在于，活动结束后我有机会在他们的任天堂博物馆玻璃展示柜中展出我的作品几个月，这巩固了我在视频游戏历史中的地位！事实上，我传说中的Game Boy变形金刚被展示在玻璃柜中，旁边是一件在海湾战争中受损的真实Game Boy。

关于我的目标，我曾想最终成为乐高集团自己的大师级模型搭建师，但此时我不太确定了——我的一部分可能只想做自己的事情，做平面设计师，在业余时间制作模型和动画。凭借我作为平面设计师在消费品行业工作的现实生活经验，我完全能适应公司中潜在的未来角色。毕竟，我在2015年被邀请去过他们的北美总部，我确实很享受在那里短暂访问的时光！

Ann： 能跟我们说说你的乐高Ideas项目——一个可活动的Bob Ross画家吗？

Julius： 关于Bob Ross创作，这是我原计划在2019年底/2020年初纳入砖块动画的一个小东西，但由于后勤问题取消了，改用于一个微缩场景。然后当然我把它提交给了乐高Ideas作为潜在的零售套装。到目前为止它已经获得了一些支持，但让我们把它推到10,000名支持者，以纪念已故伟大的Bob Ross！

Ann： 你最喜欢的官方乐高套装主题是什么？

Julius： 我最喜欢的主题是老城堡和太空套装，尤其是从1980年代末到1990年代中期的。作为一个孩子，我会无条件地玩大多数乐高套装，但我偏爱中世纪主题和未来主义科幻主题。特别是我喜欢Futurons、Blacktron、M-Tron、太空警察、冰行星2002和Spyrius——至于城堡系列我喜欢森林人、黑骑士和龙之大师，特别是作为玩具。现在作为成年人，我的右臂上有两个不同的乐高人仔纹身：一个红色Futuron宇航员和一个绿色森林人弓箭手！

Ann： 你在乐高之外还有其他爱好和兴趣吗？

Julius： 至于我的其他爱好，我在业余时间做平面设计和摄影，近年来做了很多视频工作。我偶尔也画一些画——事实上，我甚至为我的乐高摄影画过背景。我很久以前也搞过电子制作，但最近没有做新的电子项目；希望不久的将来我会重新摆弄电子技术，也许做一些带灯光和声音的电动项目。

Ann： 非常感谢这次精彩的采访，Baron von Brunk！我真的很享受研究你不可思议的乐高实验，并且很有趣能够与BrickNerd社区详细分享它们！

附言： 我最喜欢的经典视频游戏里的这个角色绝对是我的最爱！
