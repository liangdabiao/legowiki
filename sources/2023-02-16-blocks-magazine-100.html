100期乐高之爱：与 Blocks 杂志 Graham Hancock 对话

一句话总结这篇文章的核心：Blocks 乐高杂志创始人 Graham Hancock 访谈，庆祝发行100期

完整内容

Blocks 杂志正在庆祝发行100期，因此今天我们认为这是一个非常好的时机，来关注乐高社区中少数几位长期讲述乐高故事的人之一——Graham E. Hancock。他是 Blocks 的编辑，已经为这本杂志工作了92期。让我们开始这次访谈，祝贺团队达成这一里程碑，并欢迎 Graham 成为 BrickNerd 社区的最新嘉宾。

**Ann：**嗨，Graham！我们很高兴能采访一位对乐高充满热情、甚至不记得没有积木的生活是什么样的专家！请简单介绍一下你自己。

**Graham：**非常感谢你，Ann，能被 BrickNerd 邀请聊天真的很开心。我住在伦敦，正如你所说，我一生都是乐高粉丝。我差点陷入”黑暗时代”，但幸好乐高星球大战出现了，阻止了这一切。我们都知道，一个主题只是一个通往其他许多主题的入口，通过星球大战，我开始收集现代城市系列、模块化建筑、其他大型精美套装等等……

**Ann：**你对乐高的热情是从什么开始的？

**Graham：**海盗，啊呀！我小时候绝对热爱乐高海盗系列。英国有一档关于海盗的奇怪儿童电视节目，我只有模糊的记忆，我们去迪士尼世界度假时，《加勒比海盗》让我大开眼界，乐高海盗完美地融入了这种普遍的着迷之中。我对海盗的真实历史几乎不感兴趣——我非常痴迷于动作冒险的氛围和故事讲述的可能性。

我很幸运在圣诞节和生日收到了许多乐高海盗套装，所以客厅经常变成大海，各种船只和岛屿遍布其中。然后会有史诗般的战斗，到处飞着乐加农炮弹！

**Ann：**谈到 Blocks，祝贺你们发行第100期！我们知道长篇故事创作需要付出多少努力。你们在这期特别版中安排了什么内容来庆祝这个时刻？

**Graham：**因为 Blocks 是为真正的乐高粉丝准备的，所以在这期特别版中，我们想采访一位真正的乐高粉丝。我们采访了 Jamie Berard，谈到了他从童年的热爱到成年粉丝，再到成为乐高设计师，然后成为设计经理，如今成为 LEGO Masters 上的知名面孔的旅程。

我们还希望认可乐高套装的伟大历史，所以我们对 Niels Milan Pedersen 进行了采访，他分享了一些早期为海盗系列开发零件时的草图。

我挑战了我们驻场的搭建专家 Simon Pickard，让他想出100种有创意的连接乐高积木的方法——他真的做到了！虽然 BrickNerd 的读者可能会认出其中一些，但我保证至少有几个是全新的。希望这些技巧和窍门能对读者下一次创作有所帮助。

第100期还有很多其他内容，但那些是一些亮点！

**Ann：**杂志上的创意工作对你来说意味着什么？是什么驱使你与志同道合的创作者分享你对乐高的热爱？

**Graham：**我一直有写作和讲述故事的冲动。小时候，我会制作小书和杂志，用订书机装订起来。我记得有一本”书”是用打孔器打好洞、放入 LEGOLAND 活页夹里的。

当我第一次看到 Blocks 杂志时，我立刻意识到了这种媒介所提供的潜力——在印刷品中有非常独特和富有创意的内容创作方式，我想要探索这些。当时我也感觉，围绕乐高爱好的很多内容是从非常粉丝中心的角度撰写的，没有考虑乐高集团某些决定的背景。我热衷于做的是将事情置于背景中，这样粉丝社区至少能理解乐高集团为什么会以某种方式做某些事情。

在杂志上工作了八年（时间过得真的很快），驱动我的是为粉丝社区提供有价值的东西。为此，我们目前正在对杂志进行全面重新设计，这会真正使内容焕然一新——我非常期待今年晚些时候分享这个新设计。

**Ann：**我们在 BrickNerd 认可并庆祝这种热情和奉献。八年来100期对杂志来说是一个巨大的里程碑！你能告诉我们一些 Blocks 的历史吗？

**Graham：**对我们来说这确实是一件大事，那是肯定的。没人知道时间都去哪了！为此，我们在第100期中做了一个很好的回顾专题，挑选了一些我们最喜欢的期刊，并揭示了关于我们如何创作它们的一些内幕，或者它们如何代表了杂志的演变。

Blocks 杂志试刊号

简略的历史是：Blocks 杂志于2014年创刊；由 Mark Guest（编辑）、Kev Leaper 和 Will Holman（出版商）创办。我记得在英国最大的杂志零售商 WHSmith 里看到第一期时，非常高兴终于有了一本为乐高粉丝准备的精美印刷杂志。

几个月后，杂志需要一个全新的创意团队，就在那时，我和 Rob Paton、Daniel Konstanski 以及 Chris Wharfe 一起加入了 Blocks。Simon Pickard 此时已经参与了（事实上，他从试刊号起就一直参与杂志的工作）。在一些优秀撰稿人的帮助下，这组人真正设定了 Blocks 的方向和基调。

我记得第一次和团队其他成员交谈时，我感到非常幸运，因为很明显我们都有完全相同的目标——每个月提供我们能做的最好的杂志，达到尽可能高的标准。我们找到了一种很好的方式来确保杂志的内容涵盖不同类型的乐高粉丝——搭建者、收藏者、现代主题粉丝、怀旧粉丝、玩具摄影师等。

几年前 Silverback Publishing 收购了 Blocks 杂志，这是一个在正确时刻发生的巨大改变。他们制定了策略，使我们在疫情期间能够继续出版（一些比 Blocks 大得多的杂志实际上暂停了印刷或改为双月刊），并且一直非常支持我们所做的一切。

**Ann：**这确实需要团队合作，不是吗？杂志中有很多很棒的常设专栏。你能告诉我们你最喜欢的是哪个吗？

**Graham：**选择最喜欢的是太难了！“改造小队”（Mod Squad）很棒，因为它鼓励你从你的收藏中拿出一套套装并对其进行升级。以我自己的经验来说，对于一个”非搭建者”来说，鼓起勇气进行创意搭建通常令人生畏，所以改造小队提供了一种可及的方式来实现这一点。它还为套装增加了更多价值，因为不仅仅是搭建它然后放在架子上，你可以在改进它的过程中继续享受它。

我真的很喜欢”技巧”（Technique）专栏，因为 Simon 的创造力和找到连接事物新方法的能力总是让我感到惊喜。这是一个已经运行了很长时间的专栏，我们目前正在策划一个新版本。当我们推出这个全新版本时，它将提供更清晰的教程，教你如何将这些有趣的（有时具有挑战性的）元素使用方法融入到你的搭建中。

至于我写的专栏，我喜欢那些访谈，因为它们非常多样化。一个月我可能在和一位牛津大学的教授谈论古罗马，下个月 LEGO Masters 的制片人可能在解释电视制作排期是如何运作的，再下个月一位动画专家在给我讲解乐一通（Looney Tunes）。你永远不会想到在一本乐高杂志工作能接触到如此多样的主题，但正如我们粉丝所知，积木几乎触及一切。

**Ann：**主题和兴趣的多样性正是使乐高社区如此广阔和有趣的原因。我很好奇，哪些期刊对你来说最特别？

**Graham：**有几个立刻浮现在脑海中。最近一期是我们的”90年乐高”特刊，作为第90期杂志恰好在乐高集团90周年纪念期间推出，效果非常好。我觉得在周年纪念时，内容应该聚焦于公司的过去，所以我们采访了在1980年代和1990年代在比隆工作的设计师。当然，这遗漏了此前50年的历史，所以 Daniel Konstanski（《乐高积木的秘密生活》的作者）翻阅了乐高档案，找到了引人入胜的文物来讨论，并搭建了一个可爱的人仔比例的老木工店复制品。

我们关于不同乐高电影的期刊对我来说也很特别，因为它们提供了一个做一些非常不同事情的机会。我真的很喜欢电影，所以本质上我可以在做乐高内容的同时做一些电影新闻报道。对于其中一些电影，没有详细的”制作揭秘”书籍，所以我们的杂志提供了了解它们如何完成的最全面资源。真正为这些期刊锦上添花的是 Daniel 的搭建项目——他在第36期中搭建的幻影忍者城是一件美丽的作品。

我再来选一期，就是第67期。当 LEGO Ideas 把21322巴卡达湾海盗要塞寄给我评测时，我在模型中到处寻找瑕疵。作为一个乐高海盗的大粉丝，我下定决心不能给它一个完全正面的评测……大家只会觉得我在感情用事。但这确实是我搭建过的最好的套装；用成人的标准技术来复刻1989年的经典套装，能以两种不同的方式重新配置，而且两种方式都很出色……它绝对是出类拔萃的。我们用这个模型做了一个漂亮的封面，并与设计师进行了精彩的访谈。

**Ann：**我最喜欢的 Blocks 的部分之一是你们喜欢与读者分享鼓舞人心的故事。你有没有对你来说很有意义的故事，或者你特别自豪写过的故事？

**Graham：**这又是一个棘手的问题，但我可能会一直追溯到第15期，说说那个受乐高积木启发的假肢。乐高积木在现实世界中的使用方式，或者作为现实世界中某个想法的灵感，真正展示了我们的爱好可以远远超出系统内的创造力。我们可以将积木背后的理念应用到我们周围的世界中，造福我们的社区。

在这个特定案例中，Carlos Arturo Torres 想要为肢体有差异的儿童创造一个合适的假肢。Carlos 考虑到儿童最了解自己的需求，需要能够定制自己的假肢，所以他提出了一个原型来展示如何实现这一点。

我为我的书《乐高英雄》（LEGO Heroes）改编了那个专题作为其中一章，这本书将在五月出版。大部分章节对这本书来说是全新的，但那一章完美地说明了乐高的理念，所以将其纳入是有意义的。

**Ann：**你自己搭建多吗？我知道你提到过喜欢整理你的乐高套装收藏。你最喜欢的搭建主题是什么？

**Graham：**我不太确定我是否真的享受整理我的收藏！我最喜欢的主题是海盗，因为如我所说，它对我童年来说太特别了。星球大战是我必须热爱的，因为我还在买那些套装……公平地说，尽管这个主题已经运行了20多年，团队还是在推出一些惊喜，比如 75313 AT-AT。

在 Blocks 杂志工作对我来说是”危险的”，因为我被吸引去关注许多我以前忽视的主题，这意味着我收集了比我原本可能会收集的更多的套装和主题。齐天大圣（Monkie Kid）非常出色，迪士尼主题乐园套装很棒（我希望我们能获得更多人仔比例的），而 Ideas 则不断推出精彩的东西。

简短的回答是：我有太多最喜欢的主题了！

**Ann：**你参与你当地的乐高社区吗？你参加乐高大会和展览吗？

**Graham：**显然，过去几年所有的实体大会和展览都暂停了，这是一个奇怪的世界。在这个不寻常的时期，我参与了 Bricks at Home，这是 Brick Alley 和 London AFOLs 合作举办的一个虚拟活动，在此期间我在直播中采访搭建者，了解他们的作品。

我喜欢参加活动，亲自见到人们，欣赏我只在网上看过的搭建作品，或者希望是从来没见过的作品。我很快会去英国的曼彻斯特参加 Bricktastic，并希望今年能去 Skærbæk。

杂志每月截稿日期的挑战之一是总有截稿日在逼近，所以有时尽管非常想在某个活动中出席，但时间安排上往往不凑巧。这对我的创意搭建来说也是如此——尽管我已经试探性地自己搭建了几次东西，但我的创意精力往往都倾注在了 Blocks 上，所以搭建项目只能留在图纸上，或者永远无法完成。

Graham E. Hancock 和 Daniel Konstanski 在丹麦比隆的乐高之家。

**Ann：**你想谈谈乐高以外的其他爱好和兴趣吗？

**Graham：**我真的很喜欢脱口秀，住在伦敦对此来说再完美不过了。事实上，如果有国际读者访问伦敦，我强烈推荐如果有时间去看看脱口秀。一张门票比西区演出便宜得多，你可以在一些非常私密的场馆看到真正高质量的喜剧演员。电影院也是我的一个巨大兴趣，我喜欢旅行。还好我没有交友档案，因为这听起来太普通了！

**Ann：**非常感谢你接受采访，Graham！再次祝贺，我们祝愿你和团队在接下来的100期中继续创造灵感和创意！
